# VALOHELL — Valorant Takım Simülatörü

Tek dosyalık tarayıcı oyunu (`valorant.html`). Kadronu kur, taktiğini belirle,
botlara karşı tur-tur maç simüle et, lider tablosuna tırman.

## Nasıl oynanır?
`valorant.html` dosyasına **çift tıkla** — tarayıcıda açılır, sunucu gerekmez.

## Özellikler
- Online (botlara karşı) mod · 3v3 / 5v5
- Reroll'lü kadro kurma (takım/sınıf değiştir) · FC26 tarzı kimya
- Tur-tur maç simülasyonu · ekonomi · devre arası strateji
- Günlük + Tüm Zamanlar lider tabloları (tarayıcıda localStorage)
- Açık/koyu tema

## Notlar
- Bayraklar internetten (flagcdn.com) yüklenir; çevrimdışıysan ülke kodu görünür.
- Takım logoları `logos/` klasöründedir.
- Kadrolar vlr.gg üzerinden 2026 itibarıyla güncellendi.
